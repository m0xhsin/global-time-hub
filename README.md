# global-time-hub
Interactive World Clock Dashboard built with Python and Streamlit. Select countries to view their current local time, UTC offset, day of week, and day/night status. Includes a table and 24-hour comparison bar chart for quick visual insights.

**World Clock Dashboard** lets users select countries and view their current local time, UTC offset, day of week, and whether it’s day or night. The dashboard also displays a 24-hour comparison bar chart for visual insights.

---

## 🚀 Features
- 🕒 Interactive selection of countries  
- 📊 Visual 24-hour comparison of local times  
- 🌞 Day/Night indicator  
- 📋 Table view with current time, UTC offset, and weekday  
- Fully built with **Python**, **Streamlit**, **Pandas**, and **Plotly**  

---

## 🛠️ Tech Stack
- Python 3  
- Streamlit  
- Pandas  
- Plotly
